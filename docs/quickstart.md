# Project Quickstart

Fast path to get the monorepo working locally with the minimum command set.

## Prerequisites

- Java 21 on your `PATH`
- Dart SDK installed
- `jaspr_cli` installed

## Verify the Repository

From the repository root:

```bash
./scripts/verify.sh
```

## Launch the Engine TUI

From `services/engine`:

```bash
./gradlew installDist
./build/install/go-no-go-engine/bin/go-no-go-engine tui
```

Notes:

- The logical root command is `gonogo`.
- The local launcher generated by this repository is `go-no-go-engine`.

## Run the Default CLI Flow

From `services/engine`:

```bash
./gradlew run --args="run"
```

This runs the default end-to-end pipeline and writes artifacts under `services/engine/output/`.

## Start the Operations UI

From the repository root:

```bash
./scripts/run-ops-ui.sh
```

Open:

- `http://localhost:8791`

## Start the Reports UI

From the repository root:

```bash
./scripts/run-reports-ui.sh
```

Open:

- `http://localhost:8792`
